# Clothiq - Fashion E-Commerce Platform

## Introduction
Clothiq is a fullstack fashion e-commerce platform, it build in Niên Luân Cơ Sở ngành và được review + update từ 1/6


## Features
- User authentication with JWT
- Product management
- Shopping cart
- Order management
- Admin dashboard

## Tech Stack
## Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Docker
- Swagger
## Frontend
- React 19
- Tailwindcss/postcss 4

## Architecture
Monolithi architeture
Frontend -> REST API Backend -> MySQL
